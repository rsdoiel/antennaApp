About this software

antennaApp 0.0.0

prototype of a simple tool to create and curate an Antenna like website.

Authors

  • R. S. Doiel Doiel, https://orcid.org/0000-0003-0900-6903

AntennaApp provides a simple application that supports Textcasting and Link Blogging. It is built on my experience build my Antenna personal news aggregator site and my experimental feed reader project called Skimmer. It uses the Skimmer project to create a turn key Antenna style website and curation tool that can run on common computers running Raspberry Pi OS, macOS and Windows.
