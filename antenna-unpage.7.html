unpage — remove a page record from the pages collection

SYNOPSIS antenna unpage INPUT_PATH

DESCRIPTION Removes the page record associated with INPUT_PATH from the pages collection. The HTML and Markdown files on disk are not deleted.

PARAMETERS INPUT_PATH path to the source Markdown file as it was passed to ‘page’

EXAMPLE antenna unpage about.md
