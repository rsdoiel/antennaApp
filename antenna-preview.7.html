preview — serve the site on localhost

SYNOPSIS antenna preview

DESCRIPTION Starts a local HTTP server serving the htdocs directory so you can review the generated site in a browser. The host and port are set in antenna.yaml (defaults: localhost:8000).

Press Ctrl-C to stop the server.

EXAMPLE antenna preview open http://localhost:8000
