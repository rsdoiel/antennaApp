sitemap — generate sitemap XML files

SYNOPSIS antenna sitemap

DESCRIPTION Generates a set of sitemap files (sitemap_index.xml, sitemap_1.xml, …) for all pages and posts found via antenna.yaml. Place these in the root of your htdocs directory so search engines can discover your content.
