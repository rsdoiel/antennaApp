interactive — guided action wizard

SYNOPSIS antenna interactive [ACTION [PARAMETERS]]

DESCRIPTION Starts a menu-driven session that walks through any antenna action step by step. Each parameter is explained and pre-filled from arguments already on the command line. The complete command is shown before it runs.

If ACTION is omitted a menu of all available actions is presented. Useful for learning the command syntax interactively.

ALIASES tui
